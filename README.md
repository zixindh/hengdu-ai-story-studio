# Heng Du / AI Story Studio

Live: https://zixindh.github.io/hengdu-ai-story-studio/

## Visual thesis

A cinematic portfolio carved from deep ocean blue, warm ivory type, and one coral signal color: independent, precise, and quietly ambitious.

## Content plan

1. Full-bleed proof-led hero with one inquiry action.
2. Two finished films as evidence that the workflow exists.
3. Five numbered, independently sellable offers with clear scope and pricing.
4. Compact process and a final email/Instagram conversion point.

The AI Film Launch Kit includes a public free sample and a private paid Delivery Edition. Paid fulfillment assets remain ignored by Git and are backed up separately; the public repository contains only the storefront and sample.

## Interaction thesis

- The hero image drifts slowly while copy enters in a short stagger.
- Offer rows reveal on scroll and the selected number tracks the pointer.
- Inquiry links carry the chosen offer into a prefilled email; reduced-motion users get a static experience.
